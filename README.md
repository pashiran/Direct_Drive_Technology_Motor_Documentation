# Direct Drive Technology Motor Documentation

This repository contains a comprehensive collection of motor datasheets, technical documentation, and resources from Direct Drive Technology Motor company.

## Overview

Direct Drive Technology specializes in high-precision direct drive motors and actuators. This repository serves as a centralized resource for product documentation, 3D models, datasheets, and user guides.

## Repository Contents

The repository includes documentation for various motor models, organized by model number:

### Motor Models
- **M0601 Series**: Compact direct drive motors with various configurations (111, 411 variants)
- **M0602 Series**: Mid-range motors with enhanced specifications (112 variant)
- **M0603 Series**: High-performance motors in multiple configurations (A, B, C variants)
- **M0701 Series**: Advanced motor systems
- **M1502 Series**: Heavy-duty motors with extended capabilities (D, E variants)
- **M1505 Series**: Specialized motor solutions (A, B variants)
- **P1010 Series**: Precision actuators for demanding applications

### Documentation Structure

Each motor model folder typically contains:
- **3D Models**: CAD files in STEP format for mechanical integration
- **Datasheets**: Technical specifications and performance characteristics
- **User Guides**: Setup instructions, command sets, and example code
- **Configuration Files**: UART communication settings and command configurations

## Additional Resources

- Motor Control Board 0805 documentation with open-source repository links
- Simulator base motor specifications
- Product website links and technical support resources

## Usage

Navigate to the specific motor model folder to access:
1. 3D models for CAD integration
2. Datasheets for specifications
3. User guides with example code and configuration files
4. Adapter board information (where applicable)
